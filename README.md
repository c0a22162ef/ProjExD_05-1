# 逆襲！エイリアン
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
演習で実装したものに追加の機能を実装したものである。

## ゲームの実装
###共通基本機能
* 演習で実装した部分（musou_kokaton）
### 担当追加機能
* 打てない時間を作る(5発ビームをだしたら、5秒打てなくなる。打てない時間をカウントし、表示する)
